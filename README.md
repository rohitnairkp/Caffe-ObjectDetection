# Object Detection - Caffe
Object Detection using Deep Learning caffe models.
Model - **MobileNetSSD**

# Usage

python objectdetection.py --prototxt MobileNetSSD_deploy.prototxt.txt --model MobileNetSSD_deploy.caffemodel
